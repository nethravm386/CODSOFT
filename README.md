# CODSOFT Data Science Internship

This repository contains the projects completed as part of the CODSOFT Data Science Internship. The projects demonstrate the application of data analysis, visualization, and machine learning techniques to solve real-world problems.

## Projects

### Task 1: Iris Flower Classification

Built a machine learning model to classify Iris flowers into Setosa, Versicolor, and Virginica species based on sepal and petal measurements.

### Task 2: Titanic Survival Prediction

Developed a machine learning model to predict passenger survival on the Titanic using features such as age, gender, passenger class, fare, and family information.

### Task 4: Sales Prediction Using Python

Developed a Linear Regression model to predict product sales based on advertising expenditures across TV, Radio, and Newspaper platforms. Performed data analysis, visualization, model training, and evaluation using regression metrics.
